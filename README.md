## 📱 Install
1. Sideload APK (Wear OS 3.0+).
2. Import vault (QR/JSON).
3. Add tiles: Long‑press watch face → "Outlaw 2FA".

---

## 🔒 Security
- **Tink StreamingAEAD** (AES256‑GCM‑HKDF).
- Encrypted DataStore (MasterKey).
- Ghost migration from WristKey WFS.
